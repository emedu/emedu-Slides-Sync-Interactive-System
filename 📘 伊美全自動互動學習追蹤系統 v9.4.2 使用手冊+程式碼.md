# **📘 伊美全自動互動學習追蹤系統 v9.4.2 使用手冊**

---

## **✨ 系統簡介**

本系統用於 **自動化學習活動追蹤**，核心功能：

* 自動建立各階段 Google 表單並綁定活動資料簿。

* 每一輪活動 → 自動建立 **獨立的活動資料簿**（內含《學員資料總表》《進度追蹤表》）。

* 學員作答自動寫入活動資料簿，而非主控台。

* 多題型支援（單選、多選、簡答、段落）。

* 自動評分（單選/簡答完全比對，多選題支援部分得分）。

* 學員進度追蹤、提醒信件、封存資料。

* **手動分批生成表單**，避免超時或 API 配額問題。

* Email 僅需在「指定階段」收集，其餘僅輸入學號。

* **新增功能 v9.4.2**：

  * 🧹 觸發器／映射清理工具（防止失效表單殘留）。

  * 📦 一鍵「封存本活動 \+ 建立新活動」。

  * ⏱️ 防秒刷（同學號+階段，60 秒內只記錄一次）。

  * 📥 Email 收集改為「指定階段標籤」，更直覺。

  * 🏷️ 同日重交可選「保留最高分」。

  * 📁 建立的表單與資料簿自動移出 My Drive 根目錄。

  * ✅ 設定檢查再加：同階段表單標題一致、多選題必須有選項。

  * 📝 活動 Log 工作表，自動記錄警示或錯誤。

---

## **⚙️ 使用步驟**

### **1️⃣ 安裝主控台（一次性）**

1. 建立新的 Apps Script 專案。

2. 貼上 v9.4.2 程式碼。

3. 執行 `install_ControlSheet()`。

4. 系統會在雲端建立 **主控台試算表**。

---

### **2️⃣ 建立新活動**

1. 在主控台上方選單 → `👑 超級管理員` → **🆕 建立新活動**。

2. 輸入活動名稱（例：2025 秋季丙級美容）。

3. 系統會建立一個 **活動資料簿**（含學員資料總表、進度追蹤表）。

4. 所有題目設定快照會同步到活動資料簿中。

---

### **3️⃣ 設定題目**

1. 打開主控台試算表 → 《系統設定》。

2. 每一列都要填「階段標籤」與「表單標題」。

3. 設定題目內容：

   * 問題、答案欄位名稱（需唯一）、題目類型、選項、標準答案、分數。

   * 可填「截止日」「必修題數」。

4. ⚠️ 注意：

   * 「答案欄位名稱」不可重複。

   * 「多選題」必須有選項。

   * 同一階段內表單標題必須一致。

---

### **4️⃣ 分批生成表單**

1. 在主控台 → `👑 超級管理員` → **🔄 處理下一批整合表單**。

2. 每次最多建立 3 個階段。

3. 重複操作直到完成，所有表單 URL 會回填到設定表。

---

### **5️⃣ 學員作答流程**

* 學員輸入「學號」（僅第一次填指定階段需 Email）。

* 答案自動寫入當前活動的 **學員資料總表**。

* 系統自動計分並打勾 ✔ 完成狀態。

* 重複提交時，若啟用「保留最高分」，只記錄最佳成績。

---

### **6️⃣ 管理功能**

* **📧 發送提醒郵件** → 通知未完成學員。

* **🔒 封存鎖定目前活動表單** → 表單不再接受作答。

* **📦 封存本活動 \+ 建新活動** → 一鍵封存並開始新一輪。

* **🧹 觸發器／映射清理** → 移除失效表單與觸發器。

* **🧪 檢查設定** → 驗證設定表是否正確。

* **🗂️ 指定活動資料夾** → 將所有表單/資料簿移到指定資料夾。

---

## **🔑 最佳實務**

1. 每次換題目前 → 先封存舊活動（確保資料不受影響）。

2. 學號為唯一識別，Email 僅輔助。

3. 不要合併儲存格，每列完整輸入。

4. 需要跨活動統計時，可做一份「彙總報表」連結各活動資料簿。

5. 若懷疑表單題目被手動改動 → 檢查活動資料簿的 `Log` 工作表。

---

## **📝 FAQ**

**Q1. 封存後學生還能繼續作答嗎？**  
 A1. 如果你用「封存鎖定」，學生不能再提交；如果只是建立封存檔，表單仍會寫入該活動資料簿。

**Q2. 修改系統設定會影響舊活動嗎？**  
 A2. 不會。舊活動有自己的快照和資料簿，修改設定只會影響新活動。

**Q3. 可以同時管理多個活動嗎？**  
 A3. 可以，每個活動都有自己的獨立資料簿；主控台只是管理器。

 `* 🗂️ 1. Script Properties 命名空間工具`

 `* 命名規則：v94:<masterId>:<key>`

 `* 例：v94:MASTER123:run:RID123:ssId  → 值：<活動資料簿ID>`

 `**************************************************************************************************/`

``function PKEY(masterId, key) { return `v94:${masterId}:${key}`; }``

`function setProp(masterId, key, value) { PropertiesService.getScriptProperties().setProperty(PKEY(masterId, key), String(value)); }`

`function getProp(masterId, key, def = null) {`

  `const v = PropertiesService.getScriptProperties().getProperty(PKEY(masterId, key));`

  `return v == null ? def : v;`

`}`

`function delProp(masterId, key) { PropertiesService.getScriptProperties().deleteProperty(PKEY(masterId, key)); }`

`function allProps() { return PropertiesService.getScriptProperties().getProperties(); }`

`// 依 formId 在所有 properties 中尋找對應 masterId（優先使用 form:<id>:masterId）`

`function findMasterIdByFormId(formId) {`

  `const props = allProps();`

  `for (const k in props) {`

    `// 目標鍵格式：v94:<masterId>:form:<formId>:masterId`

    ``if (k.startsWith("v94:") && k.endsWith(`:form:${formId}:masterId`)) {``

      `const parts = k.split(":");`

      `return parts[1]; // masterId`

    `}`

  `}`

  `return null;`

`}`

`function process_ActiveRun_Forms_in_Batches() {`

  `const master = SpreadsheetApp.getActiveSpreadsheet();`

  `const masterId = master.getId();`

  `const runId = getProp(masterId, "activeRunId", null);`

  `if (!runId) { SpreadsheetApp.getUi().alert("尚未指定目前活動ID。請先建立新活動，或使用『🔁 指定目前活動ID』。"); return; }`

  ``const actId = getProp(masterId, `run:${runId}:ssId`, null);``

  `if (!actId) { SpreadsheetApp.getUi().alert("找不到目前活動的資料簿ID。"); return; }`

  `const actSs = SpreadsheetApp.openById(actId);`

  `const snapshot = actSs.getSheetByName(CONFIG.ACT_SETTINGS_SNAPSHOT);`

  `const allQs = Utils.readConfig(snapshot, /*verbose*/true); // 從快照讀，固定不受主控台變動影響`

  `if (allQs.length === 0) { SpreadsheetApp.getUi().alert("活動快照無有效題目。"); return; }`

  `const stageInfo = Utils.buildStageInfo(allQs);`

  `const totalStages = stageInfo.length;`

  ``let currentIndex = parseInt(getProp(masterId, `run:${runId}:batchIndex`, "0"), 10);``

  `if (currentIndex === 0) {`

    `// 初次進入此活動的批次流程：清除舊觸發器（此活動曾中斷再跑也不會掛多個）`

    `ScriptApp.getProjectTriggers().forEach(t => {`

      `if (t.getHandlerFunction() === "onFormSubmit_RunV94") ScriptApp.deleteTrigger(t);`

    `});`

  `}`

  `const endIndex = Math.min(currentIndex + CONFIG.BATCH_SIZE, totalStages);`

  ``console.log(`Run=${runId} | Stage batch: ${currentIndex} → ${endIndex} / ${totalStages}`);``

  `for (let i = currentIndex; i < endIndex; i++) {`

    `const { label: stageLabel, qs: questionsInStage } = stageInfo[i];`

    `const firstRow = questionsInStage[0].rowIndex;`

    `const isEmailStage = (stageLabel === CONFIG.REQUIRE_EMAIL_STAGE_LABEL);`

    `// 先嘗試重用（讀快照第 14 欄 formId；否則第 12 欄 URL）`

    `let form = null; let reused = false;`

    `const existingFormId = Utils.findExistingFormIdInSnapshot(snapshot, firstRow);`

    `if (existingFormId) {`

      `form = FormApp.openById(existingFormId);`

      `reused = true;`

      `// 確保 formId→masterId 映射存在`

      ``setProp(masterId, `form:${existingFormId}:masterId`, masterId);``

      ``setProp(masterId, `form:${existingFormId}:runId`, runId);``

      ``setProp(masterId, `form:${existingFormId}:stageLabel`, stageLabel);``

    `} else {`

      `form = FormApp.create(questionsInStage[0].formTitle);`

      `const formIdNew = form.getId();`

      `// 記錄 form 對應 masterId / runId / stageLabel`

      ``setProp(masterId, `form:${formIdNew}:masterId`, masterId);``

      ``setProp(masterId, `form:${formIdNew}:runId`, runId);``

      ``setProp(masterId, `form:${formIdNew}:stageLabel`, stageLabel);``

`const Scoring = {`

  `toHalfWidth: s => String(s||"").replace(/[\uFF01-\uFF5E]/g, ch => String.fromCharCode(ch.charCodeAt(0)-0xFEE0)).replace(/\u3000/g," "),`

  `normalize: s => Scoring.toHalfWidth(s).trim().toLowerCase(),`

  `splitOptions: s => String(s||"").split(",").map(x=>Scoring.normalize(x)).filter(Boolean),`

  `normalizeAnswerForStore: resp => Array.isArray(resp) ? resp.join(", ") : String(resp||""),`

  `scoreAnswer: function(q, rawResp) {`

    `let score = 0;`

    `if (q.type === "單選題" || q.type === "簡答題") {`

      `const ans = Scoring.normalize(rawResp), std = Scoring.normalize(q.standardAnswer);`

      `if (std && ans === std) score = q.score;`

    `} else if (q.type === "多選題") {`

      `const respArr = Array.isArray(rawResp) ? rawResp.map(Scoring.normalize) : Scoring.splitOptions(rawResp);`

      `const stdArr = Scoring.splitOptions(q.standardAnswer);`

      `const respSet = new Set(respArr), stdSet = new Set(stdArr);`

      `if (CONFIG.CHECKBOX_SCORING_MODE === "exact") {`

        `const sameSize = respSet.size === stdSet.size; let allIn = true; respSet.forEach(x=>{ if(!stdSet.has(x)) allIn=false; });`

        `if (sameSize && allIn) score = q.score;`

      `} else if (CONFIG.CHECKBOX_SCORING_MODE === "partialStrict") {`

        `let anyWrong = false; respSet.forEach(x=>{ if(!stdSet.has(x)) anyWrong=true; });`

        `if (!anyWrong && stdSet.size>0) {`

          `let correct=0; respSet.forEach(x=>{ if(stdSet.has(x)) correct++; });`

          `score = Math.round((correct/stdSet.size)*q.score);`

        `}`

      `} else if (CONFIG.CHECKBOX_SCORING_MODE === "partialNoPenalty") {`

        `if (stdSet.size>0) {`

          `let hit=0; respSet.forEach(x=>{ if(stdSet.has(x)) hit++; });`

          `score = Math.round((hit/stdSet.size)*q.score);`

        `}`

      `} else { // partialWithPenalty`

        `const totalStd = stdSet.size;`

        `if (totalStd>0) {`

          `let correct=0, wrong=0;`

          `respSet.forEach(x=>{ if(stdSet.has(x)) correct++; else wrong++; });`

          `const raw = (correct/totalStd) - (wrong/Math.max(respSet.size,1));`

          `score = Math.max(0, Math.round(raw*q.score));`

        `}`

      `}`

    `}`

    `return {score};`

  `}`

`};`

  `const master = SpreadsheetApp.getActiveSpreadsheet();`

  `const runId = getProp(masterId, "activeRunId", null);`

  `const totalStages = stageInfo.length;`

  ``let currentIndex = parseInt(getProp(masterId, `run:${runId}:batchIndex`, "0"), 10);``

  `if (currentIndex === 0) {`

    `// 初次進入此活動的批次流程：清除舊觸發器（此活動曾中斷再跑也不會掛多個）`

    `ScriptApp.getProjectTriggers().forEach(t => {`

      `if (t.getHandlerFunction() === "onFormSubmit_RunV94") ScriptApp.deleteTrigger(t);`

    `});`

  `}`

  ``console.log(`Run=${runId} | Stage batch: ${currentIndex} → ${endIndex} / ${totalStages}`);``

  `for (let i = currentIndex; i < endIndex; i++) {`

    `const { label: stageLabel, qs: questionsInStage } = stageInfo[i];`

    `const firstRow = questionsInStage[0].rowIndex;`

    `const isEmailStage = (stageLabel === CONFIG.REQUIRE_EMAIL_STAGE_LABEL);`

    `// 先嘗試重用（讀快照第 14 欄 formId；否則第 12 欄 URL）`

    `let form = null; let reused = false;`

    `const existingFormId = Utils.findExistingFormIdInSnapshot(snapshot, firstRow);`

    `if (existingFormId) {`

      `form = FormApp.openById(existingFormId);`

      `reused = true;`

      `// 確保 formId→masterId 映射存在`

      ``setProp(masterId, `form:${existingFormId}:masterId`, masterId);``

      ``setProp(masterId, `form:${existingFormId}:runId`, runId);``

      ``setProp(masterId, `form:${existingFormId}:stageLabel`, stageLabel);``

    `} else {`

      `form = FormApp.create(questionsInStage[0].formTitle);`

`function onFormSubmit_RunV94(e) {`

  `const lock = LockService.getScriptLock();`

  `if (!lock.tryLock(15000)) { console.log("⚠️ 無法取得鎖定"); return; }`

  `try {`

    `if (!e || !e.source || !e.response) { console.log("⚠️ 無效事件"); return; }`

    `const form = e.source;`

    `const formId = form.getId();`

    `// 優先用 formId → masterId 映射`

    `let masterId = findMasterIdByFormId(formId);`

    `// 後備：掃描 activeRunId（相容舊版；多主控時可能不準，僅做 fallback）`

    `if (!masterId) {`

      `const props = allProps();`

      `for (const k in props) {`

        `if (k.startsWith("v94:") && k.endsWith(":activeRunId")) {`

          `masterId = k.split(":")[1];`

          `break;`

        `}`

      `}`

    `}`

    `if (!masterId) { console.error("❌ 找不到 masterId"); return; }`

    ``const runId = getProp(masterId, `form:${formId}:runId`, null);``

    `if (!runId) { console.error("❌ 此表單未綁定 runId: " + formId); return; }`

    ``const actId = getProp(masterId, `run:${runId}:ssId`, null);``

    `if (!actId) { console.error("❌ 找不到 run 對應的活動資料簿ID: " + runId); return; }`

    `const actSs = SpreadsheetApp.openById(actId);`

    `const snapshot = actSs.getSheetByName(CONFIG.ACT_SETTINGS_SNAPSHOT);`

    `const allQs = Utils.readConfig(snapshot, /*verbose*/false);`

    ``const stageLabel = getProp(masterId, `form:${formId}:stageLabel`, "");``

    `const questionsInStage = allQs.filter(q => q.label === stageLabel);`

    `// 秒刷防護：同學號 + 階段 within SUBMIT_GUARD_SECONDS`

    `function withSubmitGuard_(sid, stageLabel, fn) {`

      ``const key = `v94submit:${sid}:${stageLabel}`;``

      `const cache = CacheService.getScriptCache();`

      `if (cache.get(key)) return false; // 阻擋`

      `cache.put(key, "1", CONFIG.SUBMIT_GUARD_SECONDS);`

      `try { fn(); } finally { cache.remove(key); }`

      `return true;`

    `}`

    `function getByTitle(title) {`

      `const items = e.response.getItemResponses();`

      `for (let i = 0; i < items.length; i++) {`

        `if (items[i].getItem().getTitle() === title) return items[i].getResponse();`

      `}`

      `return "";`

    `}`

    `const studentId = Utils.normalizeShort(getByTitle("你的學號是？"));`

    `if (!studentId) { console.log("❌ 學號為空，本次提交略過"); return; }`

    `// 健檢：快照題目是否仍存在於目前表單`

    `const formItemTitles = form.getItems().map(it => it.getTitle());`

    `const missing = questionsInStage.filter(q => !formItemTitles.includes(q.question));`

    `if (missing.length) {`

      `logToActivity_(actSs, "WARN", studentId, "表單題目與快照不一致：缺少→ " + missing.map(m=>m.question).join(", "));`

    `}`

    `withSubmitGuard_(studentId, stageLabel, () => {`

      `let email = Utils.normalizeEmail(getByTitle("你的Email是？（首次填一次即可）"));`

      `if (!email) email = Utils.lookupEmailByStudentId(actSs, studentId) || "";`

      `const dataSheet = actSs.getSheetByName(CONFIG.DATA_SHEET_NAME);`

      `questionsInStage.forEach(qc => {`

        `const rawResp = getByTitle(qc.question);`

        `const scored = Scoring.scoreAnswer(qc, rawResp);`

        `TriggerHandler._updateSingleDataCell(dataSheet, studentId, email, qc, Scoring.normalizeAnswerForStore(rawResp), scored.score);`

      `});`

      `TriggerHandler._updateTrackingSheet(actSs, studentId, allQs);`

    `});`

  `} catch (err) {`

    `console.error("❌ onFormSubmit_RunV94 出錯：" + err + "\n" + (err.stack || ""));`

  `} finally {`

    `lock.releaseLock();`

  `}`

`}`

    `const dataRow = Utils.findRow(dataSheet, studentId, sidCol);`

    `if (dataRow === -1) return;`

    `const dataRowVals = dataSheet.getRange(dataRow, 1, 1, dataSheet.getLastColumn()).getValues()[0];`

    `const stages = Utils.buildStageInfo(allQs);`

    `let completed = 0, totalScore = 0;`

    `const marks = stages.map(st => {`

      `const qs = allQs.filter(q => q.label === st.label);`

      `const req = Math.max(...qs.map(q=>q.requiredCount||0), 0);`

      `const doneFlags = qs.map(q => {`

        `const tIdx = dIdx[q.timestampColumn];`

        `return !!(tIdx && dataRowVals[tIdx-1]);`

      `});`

      `const doneCount = doneFlags.filter(Boolean).length;`

      `const ok = req > 0 ? (doneCount >= req) : doneFlags.every(Boolean);`

      `if (ok) completed++;`

      `return ok ? "✔" : "";`

    `});`

    `allQs.forEach(q => {`

      `const sIdx = dIdx[q.scoreColumn];`

      `if (sIdx) {`

        `const v = parseFloat(dataRowVals[sIdx-1]);`

        `if (!isNaN(v)) totalScore += v;`

      `}`

    `});`

    `const r = Utils.findRow(track, studentId, 1);`

    `const row = [studentId, ...marks, completed, totalScore];`

    `if (r === -1) track.appendRow(row);`

    `else track.getRange(r, 1, 1, row.length).setValues([row]);`

  `}`

`};`

 `* 🧮 13. 評分`

 `**************************************************************************************************/`

`const Scoring = {`

  `toHalfWidth: s => String(s||"").replace(/[\uFF01-\uFF5E]/g, ch => String.fromCharCode(ch.charCodeAt(0)-0xFEE0)).replace(/\u3000/g," "),`

  `normalize: s => Scoring.toHalfWidth(s).trim().toLowerCase(),`

  `splitOptions: s => String(s||"").split(",").map(x=>Scoring.normalize(x)).filter(Boolean),`

  `normalizeAnswerForStore: resp => Array.isArray(resp) ? resp.join(", ") : String(resp||""),`

  `scoreAnswer: function(q, rawResp) {`

    `let score = 0;`

    `if (q.type === "單選題" || q.type === "簡答題") {`

      `const ans = Scoring.normalize(rawResp), std = Scoring.normalize(q.standardAnswer);`

      `if (std && ans === std) score = q.score;`

    `} else if (q.type === "多選題") {`

      `const respArr = Array.isArray(rawResp) ? rawResp.map(Scoring.normalize) : Scoring.splitOptions(rawResp);`

      `const stdArr = Scoring.splitOptions(q.standardAnswer);`

      `const respSet = new Set(respArr), stdSet = new Set(stdArr);`

      `if (CONFIG.CHECKBOX_SCORING_MODE === "exact") {`

        `const sameSize = respSet.size === stdSet.size; let allIn = true; respSet.forEach(x=>{ if(!stdSet.has(x)) allIn=false; });`

        `if (sameSize && allIn) score = q.score;`

      `} else if (CONFIG.CHECKBOX_SCORING_MODE === "partialStrict") {`

        `let anyWrong = false; respSet.forEach(x=>{ if(!stdSet.has(x)) anyWrong=true; });`

        `if (!anyWrong && stdSet.size>0) {`

          `let correct=0; respSet.forEach(x=>{ if(stdSet.has(x)) correct++; });`

          `score = Math.round((correct/stdSet.size)*q.score);`

        `}`

      `} else if (CONFIG.CHECKBOX_SCORING_MODE === "partialNoPenalty") {`

        `if (stdSet.size>0) {`

          `let hit=0; respSet.forEach(x=>{ if(stdSet.has(x)) hit++; });`

          `score = Math.round((hit/stdSet.size)*q.score);`

        `}`

      `} else { // partialWithPenalty`

        `const totalStd = stdSet.size;`

        `if (totalStd>0) {`

          `let correct=0, wrong=0;`

          `respSet.forEach(x=>{ if(stdSet.has(x)) correct++; else wrong++; });`

          `const raw = (correct/totalStd) - (wrong/Math.max(respSet.size,1));`

          `score = Math.max(0, Math.round(raw*q.score));`

        `}`

      `}`

    `}`

    `return {score};`

  `}`

`};`
