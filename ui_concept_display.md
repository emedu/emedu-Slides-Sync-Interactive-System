This mockup demonstrates the "human-centric" design philosophy:
- **Clean & Welcoming**: The login screen greets the user with a smile (metaphorically) via welcoming text and soft gradients.
- **Focus**: The question interface minimizes distractions.
- **Aesthetics**: Navy Blue conveys trust, Gold adds a premium touch. Rounded corners feel friendly.
