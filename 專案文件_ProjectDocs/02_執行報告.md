## 三、遭遇困難與解決方式

| 困難 | 解決方式 |
|------|---------|
| Gemini-3.0 模型不存在 API 錯誤 | 回退至 `gemini-2.5-flash`（已驗證可用）|
| Admin CMS 課程階段列表硬編碼 | 改為呼叫後端 API 動態取得 Sheets 資料 |
| `UI_Admin.html` 程式碼區塊重複 | 清除重複代碼，重構為單一責任元件 |
| Gemini 回傳非結構化文字 | 強化 Prompt 要求回傳嚴格 JSON 格式 |
