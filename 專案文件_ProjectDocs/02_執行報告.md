## 三、遭遇困難與解決方式

| 困難 | 解決方式 |
|------|---------|
| `clasp push` 後代碼未更新 | 發現 `.clasp.json` 的 `scriptId` 錯位。更新 ID 並強制重啟 GAS 分頁後解決。 |
| Web App 顯示舊版號 | 此為 GAS 部署機制。需手動在「管理部署」中建立「新版本」並發布。 |
| 使用說明文字太雜亂 | 重構 `apiGetManual` 為 JSON sections，前端改用卡片組件渲染，視覺體驗提升。 |
