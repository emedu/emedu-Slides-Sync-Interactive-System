# 🚀 伊美系統 v10.2.0：系統網址領取指南

您說得完全正確！我們已經將系統「自動化」了。只要點擊幾下按鈕，系統就會為您產生所有必要的網址。

## 第一步：取得「網頁版網址」(Web App URL)
這是系統的「根網址」，前台與後台都靠它。
1. 在 GAS 編輯器上方點擊 **「部署」** -> **「新建部署」**。
2. 類型選擇 **「網頁應用程式」**，執行身分選您自己，誰可以存取選「所有人」。
3. 點擊「部署」後，您會得到一個 **「網頁應用程式網址」**。
   - ➜ **這就是「學員入口 (前台)」**。
   - ➜ 在網址後面加上 `?page=admin` **就是「管理中心 (後台)」**。

## 第二步：手動執行「系統初始化」
這一步會自動幫您建立一封全新的 Google 試算表（資料庫）。
1. 在 GAS 編輯器上方的工具列，在「偵錯」旁邊的下拉選單選中 **`setupSystem`**。
2. 點擊 **「執行」** (三角形圖示)。
3. 查看下方的 **「執行紀錄」**，系統會印出：
   - `✅ 系統初始化成功！`
   - `試算表網址: https://docs.google.com/spreadsheets/d/...`
   - ➜ **這就是您的「試算表管理介面」**。

## 📁 網址彙整清單
| 名稱 | 如何取得 / 組合 | 用途 |
| :--- | :--- | :--- |
| **學員入口 (前台)** | 部署後得到的 Web App URL | 學生作答、互動、看 AI 回饋 |
| **管理中心 (後台)** | Web App URL + `?page=admin` | 設定題目、看統計、診斷、看說明 |
| **試算表 (資料庫)** | 執行 `setupSystem` 的 Log 紀錄 | 修改進階設定、查看原始資料列 |
